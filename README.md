# 我家在回龙观 / Huilongguan Living Map

> 一个从孩子视角出发，重新认识回龙观附近生活与物理世界的可视化项目。

## 项目简介

**Huilongguan Living Map** 是一个关于北京回龙观地区的生活观察与可视化项目。

这个项目不只是介绍“回龙观在哪里”，也不只是统计道路、楼房、商场和公园，而是尝试从孩子的视角出发，观察一个人在这里生活时真实接触到的附近世界：家、路、树、影子、声音、气味、建筑、游乐场、公园、小动物、上学路，以及那些让孩子喜欢、害怕、好奇或停下来的地方。

项目希望回答���个朴素的问题：

> 一个孩子生活在回龙观，他/她每天实际看见、走过、听到、闻到、触摸到的世界是什么样的？

## 项目理念

城市不只是由道路、楼房和设施组成的。

对孩子来说，城市是：

- 从家门口到学校的一条路；
- 夏天能不能走在树荫下；
- 哪个路口需要大人牵手；
- 哪里可以跑、可以玩、可以坐下来；
- 哪里有鸟叫、猫、虫子、树叶和花；
- 哪里很吵、很晒、很危险；
- 哪里让人想停留、探索和再次回来。

因此，这个项目希望用地图、数据、照片、观察记录和可视化报告，把日常生活中容易被忽略的附近世界重新变得可见。

一句话概括：

> **从孩子的高度，重新认识回龙观。**

## 可能的研究问题

### 1. 我住的地方到底有多大？

- 从家出发 5 分钟、10 分钟、15 分钟能走到哪里？
- 孩子真实的生活半径有多大？
- 哪些地方看起来很近，但对孩子来说并不容易到达？
- 哪些地方是日常最常去的地点？

### 2. 这里有多少房子？

- 回龙观有多少栋建筑？
- 哪些地方建筑密度最高？
- 哪些地方高楼多，哪些地方更开阔？
- 小区、学校、商业、公园和道路如何拼成生活空间？

### 3. 孩子怎么移动？

- 从家到学校要经过几条路？
- 有多少个红绿灯、路口和停车出入口？
- 哪些路适合孩子走，哪些路需要特别小心？
- 人行道是否足够宽？有没有树荫？有没有可以停下来的地方？

### 4. 哪里可以玩？

- 附近有多少儿童游乐场、小广场、草地、公园？
- 哪些空间适合跑，哪些适合观察，哪些适合安静待着？
- 哪些空间是公共的，哪些属于小区内部？
- 孩子喜欢在哪里停留？为什么？

### 5. 夏天怎么走不晒？

- 哪些路有连续树荫？
- 哪些地方夏天特别晒？
- 儿童活动区有没有遮阴？
- 哪些路线适合夏天散步、上学或遛弯？

### 6. 回龙观听起来是什么样？

- 哪里能听到鸟叫？
- 哪里有车流声、地铁声、施工声、广场舞或学校广播？
- 早晨、中午、晚上，声音有什么不同？
- 哪些地方安静，哪些地方让人想快点离开？

### 7. 回龙观有哪些非人居民？

- 哪里有猫、鸟、昆虫、蜗牛、蚂蚁、蝉？
- 哪些树、草地或水边更容易发现小动物？
- 不同季节能观察到什么变化？

### 8. 孩子喜欢和害怕的地方在哪里？

- 最喜欢的地方是什么？
- 最害怕的地方是什么？
- 最想探索的地方是什么？
- 最无聊、最吵、最晒、最神秘的地方分别在哪里？

## 数据与调查方法

项目可以结合两类数据：

### 客观空间数据

可用于理解回龙观的基本物理结构。

可能包括：

- 建筑物轮廓
- 道路网络
- 公园与绿地
- 水系
- 学校、幼儿园、医院、商场、超市等 POI
- 地铁站、公交站
- 小区边界与出入口
- 树木、树冠、阴影、热环境

可能的数据来源：

- OpenStreetMap
- 高德地图 / 百度地图 POI
- 卫星图和遥感影像
- 政府公开数据
- 实地步行调查
- 手工标注与照片记录

### 孩子视角数据

这是项目最重要、也最有温度的部分。

可以记录：

- 孩子喜欢的地方
- 孩子害怕的地方
- 孩子觉得有趣的地方
- 孩子想停下来的地方
- 孩子想快速通过的地方
- 孩子听到的声音
- 孩子闻到的气味
- 孩子拍下的照片
- 孩子画出的路线和地图
- 孩子对地点的命名和描述

## 可视化方向

项目最终可以形成一份网页式可视化报告，包含：

- 回龙观生活地图
- 建筑轮廓地图
- 建筑密度热力图
- 5/10/15 分钟步行圈
- 上学路线与风险点地图
- 儿童游乐空间分布图
- 树荫与夏季阴凉路线图
- 声音地图
- 气味地图
- 非人居民观察地图
- 孩子喜欢 / 害怕 / 好奇地点地图
- 同一地点的季节变化照片
- 孩子手绘地图与观察日记

## 推荐的第一阶段范围

第一阶段不必覆盖整个回龙观，可以先从一个较小范围开始：

> **从家出发 1 公里的孩子生活圈**

优先回答 7 个问题：

1. 1 公里内有多少栋建筑？
2. 有多少个小区出入口？
3. 有多少个儿童可玩空间？
4. 有多少条需要小心过的路？
5. 有多少个可以看到树和小动物的地方？
6. 夏天哪条路最阴凉？
7. 孩子最喜欢和最害怕的地方在哪里？

## 技术路线设想

### 轻量版本

适合快速开始：

- 数据记录：飞书表格 / Notion / Airtable
- 地图标注：高德地图收藏 / Google My Maps / Felt
- 图表：Datawrapper / Flourish
- 页面：Notion / Framer / Webflow / GitHub Pages
- 图片：手机拍摄 + Figma 排版

### 进阶版本

适合做成独立网页：

- 地图：Leaflet / Mapbox / MapLibre
- 数据：GeoJSON
- 分析：QGIS
- 图表：Observable Plot / ECharts
- 页面：Astro / Next.js / SvelteKit

### 数据分析版本

适合更深入的空间分析：

- Python
- GeoPandas
- OSMnx
- Shapely
- Rasterio
- QGIS
- OpenStreetMap / Overpass API
- 高德地图 API
